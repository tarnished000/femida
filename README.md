# femida
Это качественная утилита для офисных сотрудников, юристов, бухгалтеров или аналитиков, которым регулярно приходится искать информацию (например, контрагентов, номера договоров или суммы) по десяткам разных Excel-таблиц одновременно. Код написан грамотно: с обработкой ошибок (try-except), логированием в консоль (print) и заботой о UX (потоки, кэш).
