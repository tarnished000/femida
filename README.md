# Femida
Это качественная утилита для офисных сотрудников, юристов, бухгалтеров или аналитиков, которым регулярно приходится искать информацию (например, контрагентов, номера договоров или суммы) по десяткам разных Excel-таблиц одновременно. Код написан грамотно: с обработкой ошибок (try-except), логированием в консоль (print) и заботой о UX (потоки, кэш, стили).

# 1. Основной функционал (Что видит и делает пользователь)
Выбор папки с файлами: Пользователь выбирает директорию, приложение сканирует её на наличие Excel-документов и загружает их в память. При перезапуске программа автоматически вспоминает последнюю открытую папку.

Два режима поиска:

Обычный (по вхождению): Ищет совпадения подстроки (без учета регистра). Например, запрос «иван» найдет и «Иванов», и «диван».

Строгий (полное совпадение): Ячейка должна полностью соответствовать запросу. Полезно, если нужно найти, например, конкретный пустой столбец или точный артикул.

Вкладки с результатами: Для каждого Excel-файла, в котором найдено совпадение, создается отдельная вкладка с таблицей.

Подсветка совпадений: Найденные ячейки окрашиваются в приятный нежно-зеленый цвет.

Фильтрация «на лету» (двойной клик): Если дважды кликнуть по заголовку любой колонки в результатах поиска, откроется небольшое окошко, куда можно ввести текст, чтобы мгновенно отфильтровать строки прямо внутри этой вкладки.

# 2. Архитектура и оптимизация (Что под капотом)
Многопоточность (QThread)
Все тяжелые операции (чтение больших файлов Excel и сам процесс поиска) вынесены в отдельный фоновый поток Worker. Благодаря этому интерфейс программы не зависает («Не отвечает»), пока лопатит гигабайты таблиц, а пользователь видит прогресс в статус-баре.

Кэширование данных (Pickle)
Чтение файлов Excel через pandas — операция относительно медленная. Чтобы ускорить повторные запуски, автор кода реализовал кэширование:

# 3. Косметика и интерфейс
В самом конце кода переменная STYLE_SHEET содержит объемный CSS-подобный код (QSS) для стилизации Qt-виджетов. Стиль настроен под тему Fusion с использованием спокойных серо-синих, голубых и белых тонов, кастомными кнопками, радио-баттонами и скругленными углами полей ввода.
